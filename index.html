gpt-from-scratch

Função

Implementação do zero de um modelo GPT em PyTorch.

Problema

Entender a arquitetura Transformer implementando cada componente manualmente, sem usar abstrações de alto nível como nn.MultiheadAttention.

Solução

Character-level language model treinado no corpus Shakespeare com Multi-Head Self-Attention implementado do zero, visualização de attention maps e geração de texto com temperature sampling e top-k filtering.

Resultado

Modelo funcional que gera texto com estrutura shakespeariana, val_loss ~1.5 no treino completo com 5000 iterações no Colab (GPU T4).

Tecnologias

Python, PyTorch, NLP, Transformers

ai-slop-detector

Função

Web app Flask para detectar texto gerado por IA.

Problema

Identificar conteúdo gerado por IA comparando abordagens de custo e acurácia diferentes num produto funcional.

Solução

DistilBERT fine-tunado no dataset HC3 rodando em paralelo com OpenAI gpt-4o-mini, scraping automático de URLs e interface de resultado lado a lado com barra de score visual.

Resultado

11 testes automatizados com pytest, detecção dupla funcional, README bilíngue com diagrama de arquitetura.

Tecnologias

Python, Flask, HuggingFace, OpenAI API
