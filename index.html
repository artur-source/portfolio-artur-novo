Projetos

RegiFlex

Destaque

Função

Desenvolvimento full-stack de um sistema de cadastro e gestão de pacientes.

Problema

Necessidade de um sistema eficiente para registro de pacientes com validação de dados e rastreabilidade via QR Code.

Solução

Implementação de backend em **PHP** com banco de dados **MySQL** para persistência. Frontend com interface responsiva e geração de **QR Code** para identificação rápida. Design e prototipagem realizados no **Figma**.

Resultado

Sistema funcional que automatiza o processo de cadastro, reduzindo erros e permitindo a consulta rápida de informações através da leitura do QR Code.

JV STORE (Catálogo Estático)

Função

Criação de um catálogo de produtos estático e responsivo.

Problema

A loja precisava de uma vitrine digital leve e de fácil manutenção, com fluxo de compra direto para o WhatsApp.

Solução

Desenvolvimento com **HTML, CSS e JavaScript** puro. Utilização de **JSON** para gerenciar os dados dos produtos, permitindo busca e filtragem em tempo real sem a necessidade de um backend complexo.

Resultado

Catálogo otimizado para mobile, com carregamento rápido e integração direta com o **WhatsApp** para finalizar pedidos, simplificando o processo de compra para o cliente.
